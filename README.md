# Aigostar — Homey app

Control **Aigostar** smart bulbs (the AigoSmart app) from Homey Pro: on/off,
brightness, warm/cool white, color and light mode, plus a Flow action to set the
color temperature by percentage.

> Unofficial, community-built app. Not affiliated with, authorised or endorsed by
> Aigostar. All trademarks belong to their respective owners.

## Pairing

Add a device → Aigostar → sign in with your AigoSmart account. If the account
requires it, a one-time email verification code is requested during pairing.
Your bulbs are discovered automatically.
